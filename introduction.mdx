## Why Tohka Theme?

Tohka offers a clean, minimalist approach to WordPress themes with a focus on performance and developer experience. Built with modern standards and best practices, it provides everything you need to create beautiful websites without the bloat.

## What's Included

- **Theme Setup Functions** - Automatic feature registration and WordPress support
- **Asset Management** - Organized CSS and JavaScript loading with cache-busting
- **Widget System** - Pre-configured sidebar with customizable widget areas
- **Template Files** - Clean header, footer, and content templates
- **Navigation Support** - Built-in menu system ready for customization
- **Post Thumbnails** - Featured image support for posts and pages
